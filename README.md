# spec-driven-development

Ideas about Spec-Driven Development.

## Presentations

This repository includes Marp presentations for explaining Spec-Driven Development concepts.

### Getting Started with Presentations

1. Install dependencies:
```bash
npm install
```

2. Preview presentations:
```bash
npm run present:preview
```

3. Export to PDF:
```bash
npm run present:pdf
```

4. Export to HTML:
```bash
npm run present:html
```

### Available Presentations

- `presentations/spec-driven-development-intro.md` - Introduction to Spec-Driven Development methodology

### Creating New Presentations

Create new `.md` files in the `presentations/` directory with Marp frontmatter:

```yaml
---
marp: true
theme: default
---
```

See `presentations/README.md` for detailed instructions.
